## ⚙️ Run Locally

Clone the project

```bash
git clone https://github.com/Kerala-Blockchain-Academy/gin-app-example.git
cd gin-app-example
```

Install dependencies

```bash
go mod download
```

Start the application

```bash
go run .
```
